# Zomato Bangalore Data Cleaning & Exploratory Data Analysis

## Project Overview

This project focuses on analyzing the Zomato Bangalore restaurant
dataset to uncover business insights, customer behavior patterns, and
pricing trends.

The project demonstrates practical data cleaning, preprocessing, and
exploratory data analysis (EDA) skills using Python.

------------------------------------------------------------------------

## Objectives

-   Perform data cleaning and preprocessing
-   Handle missing and inconsistent values
-   Remove duplicates
-   Transform rating and cost-related features
-   Analyze restaurant distribution across locations
-   Identify pricing and rating trends
-   Generate meaningful business insights using visualization

------------------------------------------------------------------------

## Tools & Technologies Used

-   Python
-   Pandas
-   NumPy
-   Matplotlib
-   Seaborn
-   Jupyter Notebook

------------------------------------------------------------------------

## Key Steps Performed

### 1. Data Cleaning

-   Removed unnecessary columns
-   Handled missing values
-   Cleaned "rate" column (removed 'NEW', '-', '/5')
-   Converted rating column to numeric
-   Removed duplicate records

### 2. Exploratory Data Analysis

-   Analyzed restaurant types
-   Examined online ordering trends
-   Explored location-wise restaurant distribution
-   Studied pricing distribution (Cost for Two)
-   Visualized rating distribution

### 3. Business Insights

-   Casual Dining and Quick Bites dominate the market
-   Majority of restaurants fall in affordable to mid-range pricing
-   Ratings mostly range between 3.5 and 4.2
-   High restaurant density in commercial areas
-   Strong presence of online ordering services

------------------------------------------------------------------------

This project highlights real-world data cleaning challenges and
